## webpack


### Configure

~~~bash
yarn add webpack webpack-bundle-tracker
~~~

~~~bash
$ mkdir -p assets/js
$ vim assets/js/main.js
~~~

### Bundle

~~~bash
$ ./node_modules/.bin/webpack --config webpack/webpack.config.js
~~~
